# Advanced Thermal Management Project

This repository contains the report and presentation for a project on Advanced Thermal Management in Power Electronics. The project focuses on cooling techniques for inverters and converters, including practical insights and analysis.

## Files Included
1) 'Report.pdf' – Detailed project report
2) 'Presentation.pptx' – Presentation slides

## Overview
1) Explores thermal challenges in power electronics systems
2) Analyzes cooling methods and strategies for inverters/converters
3) Highlights practical implications for efficient thermal management
